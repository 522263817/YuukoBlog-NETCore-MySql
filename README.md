# Yuuko Blog

A light-weight single user blog system which based on ASP.Net Core RC2

- ASP.Net Core RC2
- Entity Framework Core RC2
- SQLite v3
- ASP.Net MVC Core
- Pomelo Extensions for .Net Core RC2
